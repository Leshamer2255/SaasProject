Installation Guide

Follow these steps to get FinTrack up and running on your system.


# Install FinTrack CLI
npm install -g fintrack-cli

# Initialize your project
fintrack init

# Configure your settings
fintrack config set api-key YOUR_API_KEY

# Install FinTrack CLI
yarn global add fintrack-cli

# Initialize your project
fintrack init

# Configure your settings
fintrack config set api-key YOUR_API_KEY

# Download the latest release
curl -O https://github.com/fintrack/cli/releases/latest/download/fintrack-cli

# Make it executable
chmod +x fintrack-cli

# Move to your PATH
sudo mv fintrack-cli /usr/local/bin/

Important

Make sure to keep your API key secure and never share it publicly.

// Example: Initialize FinTrack SDK
const fintrack = new FinTrack({
    apiKey: 'YOUR_API_KEY',
    organization: 'YOUR_ORG_ID'
});

// Connect to your account
await fintrack.connect();

// Set up event listeners
fintrack.on('transaction', handleTransaction);
fintrack.on('budget-alert', handleBudgetAlert);
